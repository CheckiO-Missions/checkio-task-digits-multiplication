Загалом, ідея - конвертувати дане число number в рядок і потім брати кожен символ окремо і конвертувати його назад в число.

Добре, з чого мені почати?

Спершу ти можеш проголосити змінну res з початковим значенням 1 для збереження результатів множення. Зауваж, коли тобі потрібно накопичити результат додавання, ти використовуєш початкове значення 0, але у випадку множення це має бути 1.

По-друге, тобі потрібно конвертувати дане число number у рядок, щоб мати змогу брати цифри одну за одною. Прочитай про вбудовану функцію str(), щоб це зробити. 

res = 1
text = str(number)

Як мені брати окремі символи з рядка?

Використай цикл for для проходження по text. Оскільки ти використаєш змінну text лише раз, то тобі насправді не потрібно її проголошувати. 

res = 1
for char in str(number):
    ...

Але ж кожен символ зараз це рядок. Що із цим робити?

Так. І ти маєш перетворити кожен символ char назад в число всередині циклу. Зроби це із вбудованою функцією int(). 

res = 1
for char in str(number):
    num = int(char)
    ...

Мені потрібні лише цифри відмінні від 0. Допоможи мені їх перевірити.

Ця перевірка дуже проста з виразом if. Ти можеш написати if num != 0, але Python розумний і розглядає кожне не нульове число як логічне True. Тому тобі достатньо написати наступне. 

res = 1
for char in str(number):
    num = int(char)
    if num:
        ...

Ти можеш використати "моржевий" оператор, щоб записати два останні вирази більш компактно. Цей оператор дозволяє присвоїти вираз змінній при першому використанні. 

res = 1
for char in str(number):
    if num := int(char):
        ...

Я відчуваю, що це майже все, але моя змінна res все ще рівна 1. Як накопичити результат множення?

Якщо num не рівний 0 (всередині виразу if), помнож свій res з num і присвоюй знову до res 

res = res * num
# або
res *= num

То як має виглядати моє остаточне рішення?

Тобі лише потрібно обгорнути код у вираз функції і повернути твій res.

Ось повний розв'язок:

def checkio(number: int) -> int:

    res = 1
    for char in str(number):
        if num := int(char):
            res *= num
            
    return res

У цьому розв'язку ти не робиш ніяких перетворень у рядок, а ділиш number на 10 поки number не стане меншим 1. Множ res і остачу від ділення, якщо вона не дорівнює 0. 

def checkio(number: int) -> int:

    res = 1
    while number > 1:
        number, rem = divmod(number, 10)
        res = res * (1, rem)[bool(rem)]
        
    return res

У цьому розв'язку ти робиш генератор, щоб відфільтрувати нульові символи, з'єднуєш інші знаком множення (між кожними двома) і потім виконуєш отриманий рядок - він конвертується з рядка у вираз і робляться відповідні розрахунки. 

def checkio(number: int) -> int:

    return eval('*'.join(char for char in str(number) if char != '0'))
