Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

ogólny pomysł polega na przekonwertowaniu danego number na ciąg znaków, a następnie pobraniu każdego znaku osobno i przekonwertowaniu go z powrotem na liczbę całkowitą.

Ok, od czego powinienem zacząć?

Po pierwsze, może Pan zadeklarować zmienną res z wartością początkową 1 do przechowywania wyniku mnożenia. Proszę zauważyć, że w przypadku agregacji wyniku dodawania, należy użyć wartości początkowej 0, natomiast w przypadku mnożenia powinna to być wartość 1.

Po drugie, należy przekonwertować daną number na ciąg znaków, aby móc pobierać cyfry jedna po drugiej. W tym celu proszę zapoznać się z wbudowaną funkcją str(). 

res = 1
text = str(number)

Ok, jak uzyskać oddzielne znaki z ciągu?

Proszę użyć for loop do iteracji po text. Ponieważ użyje Pan zmiennej text tylko raz, nie musi Pan jej deklarować. 

res = 1
for char in str(number):
    ...

Ale każdy znak jest teraz ciągiem znaków. Co z nim zrobić?

Tak. Proszę przekonwertować każdy char z powrotem na liczbę całkowitą wewnątrz pętli. Proszę to zrobić za pomocą wbudowanej funkcji int(). 

res = 1
for char in str(number):
    num = int(char)
    ...

Potrzebuję tylko cyfr nie równych 0. Proszę mi pomóc je sprawdzić.

To sprawdzanie jest bardzo proste z if statement. Można napisać if num!= 0, ale Python jest sprytny i traktuje każdą niezerową liczbę jako wartość logiczną True. Można więc po prostu napisać następujące polecenie. 

res = 1
for char in str(number):
    num = int(char)
    if num:
        ...

Mogą Państwo użyć walrus operator do napisania dwóch ostatnich wyrażeń w bardziej kompaktowy sposób. Operator Walrus pozwala przypisać wyrażenie do zmiennej już w pierwszym użyciu. 

res = 1
for char in str(number):
    if num:= int(char):
        ...

Wypełniłem go prawie w całości, ale moje res wciąż jest równe 1. Jak skumulować wynik mnożenia?

jeśli num nie jest równe 0 (wewnątrz if statement), proszę pomnożyć res przez num i ponownie przypisać do res 

res = res * num
# or
res *= num

Jak powinno wyglądać moje ostateczne rozwiązanie?

Musi Pan tylko zawinąć kod w wyrażenie funkcji i zwrócić res.

Oto całe rozwiązanie:

def checkio(number: int) -> int:

    res = 1
    for char in str(number):
        if num:= int(char):
            res *= num
            
    return res

Jasne, proszę pobrać!

W tym rozwiązaniu nie dokonuje Pan konwersji na ciąg znaków, ale dzieli number przez 10, aż number stanie się mniejsze niż 1. Proszę pomnożyć res i wyjść z dzielenia, jeśli nie jest równe 0. 

def checkio(number: int) -> int:

    res = 1
    while number> 1:
        number, rem = divmod(number, 10)
        res = res * (1, rem)[bool(rem)]
        
    return res

W tym rozwiązaniu tworzy Pan generator, który odfiltrowuje znaki zerowe, łączy je ze znakiem mnożenia (między każdymi dwoma), a następnie ocenia ciąg - konwertuje go z ciągu na wyrażenie i wykonuje niezbędne obliczenia. 

def checkio(number: int) -> int:

    return eval('*'.join(char for char in str(number) if char!= '0'))
