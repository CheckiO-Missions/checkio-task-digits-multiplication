die allgemeine Idee ist, eine gegebene number in eine Zeichenkette umzuwandeln und dann jedes Zeichen einzeln zu nehmen und wieder in eine Ganzzahl umzuwandeln.

Ok, womit soll ich anfangen?

Zunächst können Sie die Variable res mit dem Anfangswert 1 deklarieren, um das Ergebnis der Multiplikation zu speichern. Beachten Sie, dass Sie, wenn Sie das Ergebnis der Addition aggregieren müssen, den Anfangswert 0 verwenden, aber im Falle der Multiplikation sollte es 1 sein.

Zweitens müssen Sie eine gegebene number in eine Zeichenkette umwandeln, damit Sie die Ziffern einzeln übernehmen können. Lesen Sie dazu die integrierte Funktion str(). 

res = 1
text = str(number)

Ok, wie bekommt man einzelne Zeichen aus der Zeichenkette?

Verwenden Sie for loop, um über text zu iterieren. Da Sie die Variable text nur einmal verwenden werden, brauchen Sie sie nicht zu deklarieren. 

res = 1
for char in str(number):
    ...

Aber jedes Zeichen ist jetzt eine Zeichenkette. Was tun Sie damit?

Ja. Und Sie müssen jedes char innerhalb der Schleife wieder in eine Ganzzahl umwandeln. Verwenden Sie dazu die integrierte Funktion int(). 

res = 1
for char in str(number):
    num = int(char)
    ...

Ich brauche nur Ziffern ungleich 0. Helfen Sie mir, diese zu überprüfen.

Diese Überprüfung ist mit if statement sehr einfach. Sie können if num!= 0 schreiben, aber Python ist schlau und betrachtet jede Zahl ungleich Null als booleschen Wert True. Sie können also einfach das Folgende schreiben. 

res = 1
for char in str(number):
    num = int(char)
    if num:
        ...

Sie können walrus operator verwenden, um die beiden letzten Ausdrücke kompakter zu schreiben. Mit dem Walrus-Operator können Sie den Ausdruck gleich bei der ersten Verwendung einer Variablen zuweisen. 

res = 1
for char in str(number):
    if num:= int(char):
        ...

Ich habe es fast vollständig ausgefüllt, aber mein res ist immer noch gleich 1. Wie kann man das Ergebnis einer Multiplikation akkumulieren?

wenn num nicht gleich 0 ist (innerhalb von if statement), multiplizieren Sie Ihr res mit num und weisen Sie es zurück zu res 

res = res * num
# or
res *= num

Wie sollte also meine endgültige Lösung aussehen?

Sie müssen den Code nur in einen Funktionsausdruck einpacken und Ihre res zurückgeben.

Hier ist die gesamte Lösung:

def checkio(number: int) -> int:

    res = 1
    for char in str(number):
        if num:= int(char):
            res *= num
            
    return res

In dieser Lösung nehmen Sie keine Umwandlung in einen String vor, sondern teilen number durch 10, bis number kleiner als 1 ist. Multiplizieren Sie res und erinnern Sie sich an die Division, wenn sie ungleich 0 ist. 

def checkio(number: int) -> int:

    res = 1
    while number> 1:
        number, rem = divmod(number, 10)
        res = res * (1, rem)[bool(rem)]
        
    return res

Bei dieser Lösung erstellen Sie einen Generator, um Nullzeichen herauszufiltern, verbinden sie mit einem Multiplikationszeichen (zwischen jeweils zwei) und werten dann die Zeichenkette aus - konvertieren sie von einer Zeichenkette in einen Ausdruck und führen die notwendigen Berechnungen durch. 

def checkio(number: int) -> int:

    return eval('*'.join(char for char in str(number) if char!= '0'))
