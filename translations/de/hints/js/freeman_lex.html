die allgemeine Idee ist, eine gegebene data in eine Zeichenkette umzuwandeln und dann jedes Zeichen einzeln zu nehmen und wieder in eine Ganzzahl umzuwandeln.

Ok, womit soll ich anfangen?

Zunächst können Sie die Variable res mit dem Anfangswert 1 deklarieren, um das Ergebnis der Multiplikation zu speichern. Beachten Sie, dass Sie, wenn Sie das Ergebnis der Addition aggregieren müssen, den Anfangswert 0 verwenden, aber im Falle der Multiplikation sollte es 1 sein.

Zweitens müssen Sie eine gegebene data in eine Zeichenkette umwandeln, damit Sie die Ziffern einzeln übernehmen können. Informieren Sie sich über die Methode String(), um dies zu tun. 

let res = 1;
let text = String(data);

Nun müssen Sie die Zeichenkette mit der Methode split() in ein Array umwandeln, dessen Argument ein leeres Zeichen ist. 

let res = 1;
let text = String(data).split("");

Ok, wie bekomme ich einzelne Zeichen aus dem Array?

Verwenden Sie for/of loop, um über das Array zu iterieren. Da Sie die Variable text nur einmal verwenden werden, müssen Sie sie nicht deklarieren. 

let res = 1;
for (let char of String(data).split("")) {
    ...}

Aber jedes Zeichen ist jetzt eine Zeichenkette. Was tun Sie damit?

Ja. Und Sie müssen jedes char innerhalb der Schleife wieder in eine Ganzzahl umwandeln. Tun Sie dies mit der Methode Number(). 

let res = 1;
for (let char of String(data).split("")) {
    let num = Number(char);
    ...}

Ich brauche nur Ziffern ungleich 0. Helfen Sie mir, diese zu überprüfen.

Diese Überprüfung ist mit if statement sehr einfach. Sie können if (num!== 0) schreiben, aber Javascript ist schlau und betrachtet jede Zahl ungleich Null als boolean true. Sie können also einfach das Folgende schreiben. 

let res = 1;
for (let char of String(data).split("")) {
    let num = Number(char);
    if (num) ...}

Ich habe fast alles ausgefüllt, aber mein res ist immer noch gleich 1. Wie kann ich das Ergebnis der Multiplikation akkumulieren?

wenn num nicht gleich 0 ist (innerhalb von if statement), multiplizieren Sie Ihr res mit num und weisen Sie es zurück zu res 

res = res * num;
// or
res *= num;

Wie sollte also meine endgültige Lösung aussehen?

Sie müssen den Code nur in einen Funktionsausdruck einpacken und Ihre res zurückgeben.

Hier ist die gesamte Lösung:

function digitsMultip(data: number): number {

    let res = 1;
    for (let char of String(data).split("")) {
        let num = Number(char);
        if (num) res*= num;
    };
    return res;
}

In dieser Lösung nehmen Sie keine Umwandlung in einen String vor, sondern teilen data durch 10, bis data kleiner als 1 ist. Multiplizieren Sie res und erinnern Sie sich an die Division, wenn sie nicht gleich 0 ist. 

function digitsMultip(data: number): number {

    let res = 1;
    while (data> 1) {
        let rem = data % 10;
        res *= rem === 0? 1: rem;
        data = Math.floor(data/10);
    };
    return res;
}

Bei dieser Lösung wird die anonyme Funktion verwendet. Die Zahl wird in einen String umgewandelt, in ein Array aufgeteilt und das Ergebnis mit der Funktion reduce ermittelt.
