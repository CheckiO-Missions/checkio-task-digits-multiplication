首先，你可以声明变量res ，其初始值为1 ，以存储乘法的结果。注意，如果你需要汇总加法的结果，你可以使用初始值0 ，但在乘法的情况下，应该是1 。

其次，你需要将给定的data 转换为字符串，以便能够逐个取数。请阅读String() 的方法来实现这一点。 

let res = 1;
let text = String(data);

现在你需要用split() 方法将字符串转换成数组，参数为空字符。 

let res = 1;
let text = String(data).split("");

好的，如何从数组中获得独立的字符？

使用for/of loop 来遍历数组。由于你将只使用一次变量text ，你实际上不需要声明它。 

let res = 1;
for (let char of String(data).split("")) {
    ...}

但现在每个字符都是一个字符串。如何处理它呢？

是的。你需要在循环中把每个char 转换为整数。用Number() 方法来做。 

let res = 1;
for (let char of String(data).split("")) {
    let num = Number(char);
    ...}

我只需要不等于0的数字，帮助我检查它们。

用if statement ，这种检查非常简单。你可以写成if (num!== 0) ，但Javascript很聪明，它把每个非零的数字看作是布尔值true 。所以你可以只写下面的内容。 

let res = 1;
for (let char of String(data).split("")) {
    let num = Number(char);
    if (num) ...}

我把它填得差不多了，但我的res 仍然等于1 。如何累积乘法的结果？

如果num 不等于0（在if statement 内），将你的res 与num 相乘，然后重新分配到res 

res = res * num;
// or
res *= num;

那么，我的最终解决方案应该是什么样子的？

你只需要将代码包装成一个函数表达式并返回你的res 。

下面是整个解决方案：

function digitsMultip(data: number): number {

    let res = 1;
    for (let char of String(data).split("")) {
        let num = Number(char);
        if (num) res*= num;
    };
    return res;
}

在这个解决方案中，你不需要转换为字符串，而是用data 除以10 ，直到data 小于1。乘以res ，如果不等于0，就从除法中提醒一下。 

function digitsMultip(data: number): number {

    let res = 1;
    while (data> 1) {
        let rem = data % 10;
        res *= rem === 0? 1: rem;
        data = Math.floor(data/10);
    };
    return res;
}
