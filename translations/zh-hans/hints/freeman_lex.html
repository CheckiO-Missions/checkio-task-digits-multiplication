好吧，我应该从什么地方开始呢？

首先，你可以声明变量res ，其初始值为1 ，以存储乘法的结果。注意，如果你需要汇总加法的结果，你可以使用初始值0 ，但在乘法的情况下，应该是1 。

其次，你需要将给定的number 转换为字符串，以便能够逐个取数。请阅读内置的str() 函数来完成这个任务。 

res = 1
text = str(number)

好了，如何从字符串中获取独立的字符？

使用for loop 来遍历text 。由于你只使用一次变量text ，所以实际上你不需要声明它。 

res = 1
for char in str(number):
    ...

但现在每个字符都是一个字符串。怎么处理它呢？

是的。你需要在循环中把每个char 转换为整数。用内置的int() 函数来做。 

res = 1
for char in str(number):
    num = int(char)
    ...

我只需要不等于0的数字，帮我检查一下。

这种检查用if statement 非常简单。你可以写if num!= 0 ，但是 Python 很聪明，它把每个非零的数字都看作布尔值True 。所以你可以直接写如下。 

res = 1
for char in str(number):
    num = int(char)
    if num:
        ...

你可以用walrus operator 来把最后两个表达式写得更紧凑。Walrus 操作符允许在第一次使用时就将表达式分配给变量。 

res = 1
for char in str(number):
    if num:= int(char):
        ...

我几乎填满了它，但我的res 仍然等于1 。如何累积乘法的结果？

如果num 不等于0（在if statement 内），将你的res 与num 相乘，然后重新分配到res 

res = res * num
# or
res *= num

那么，我的最终解决方案应该是什么样子的？

你只需要将代码包装成一个函数表达式并返回你的res 。

下面是整个解决方案：

def checkio(number: int) -> int:

    res = 1
    for char in str(number):
        if num:= int(char):
            res *= num
            
    return res

在这个解决方案中，你不需要转换为字符串，而是用number 除以10 ，直到number 小于1。乘以res ，如果不等于0就从除法中提醒。 

def checkio(number: int) -> int:

    res = 1
    while number> 1:
        number, rem = divmod(number, 10)
        res = res * (1, rem)[bool(rem)]
        
    return res

在这个解决方案中，你做一个生成器来过滤掉零字符，用乘号（在每两个字符之间）连接它们，然后评估字符串--把它从字符串转换成表达式，并做必要的计算。 

def checkio(number: int) -> int:

    return eval('*'.join(char for char in str(number) if char!= '0'))
