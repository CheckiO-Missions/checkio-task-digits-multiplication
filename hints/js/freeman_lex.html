if num is not equal 0 (inside if statement), multiply your res with num and reassign back to res 

res = res * num;

res *= num;

So, how my final solution should look like?

You just need to wrap the code into a function expression and return your res.

Here is the whole solution:

function digitsMultip(data: number): number {

    let res = 1;
    for (let char of String(data).split("")) {
        let num = Number(char);
        if (num) res*= num;
    };
    return res;
}

In this solution you do no conversion into string, but divide data by 10 until data becomes less than 1. Multiply res and reminder from the division if it's not equal 0. 

function digitsMultip(data: number): number {

    let res = 1;
    while (data > 1) {
        let rem = data % 10;
        res *= rem === 0 ? 1: rem;
        data = Math.floor(data/10);
    };
    return res;
}
