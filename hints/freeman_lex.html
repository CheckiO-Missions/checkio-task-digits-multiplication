I only need digits not equal 0. Help me to check them.

This checking is very simple with if statement. You may write if num != 0, but Python is smart and consider each non-zero number as boolean True. So you may just write the following. 

res = 1
for char in str(number):
    num = int(char)
    if num:
        ...

You may use walrus operator to write the two last expressions more compact. Walrus operator allows to assign expression to variable right in first usage. 

res = 1
for char in str(number):
    if num := int(char):
        ...

I fill it's almost all, but my res is still equals 1. How to accumulate multiplication result?

if num is not equal 0 (inside if statement), multiply your res with num and reassign back to res 

res = res * num
# or
res *= num

So, how my final solution should look like?

You just need to wrap the code into a function expression and return your res.

Here is the whole solution:

def checkio(number: int) -> int:

    res = 1
    for char in str(number):
        if num := int(char):
            res *= num
            
    return res

In this solution you do no conversion into string, but divide number by 10 until number becomes less than 1. Multiply res and reminder from the division if it's not equal 0. 

def checkio(number: int) -> int:

    res = 1
    while number > 1:
        number, rem = divmod(number, 10)
        res = res * (1, rem)[bool(rem)]
        
    return res

In this solution you make a generator to filter off zero characters, join them with multiply sign (between each two) and then evaluate the string - convert it from string to expression and do necessary computations. 

def checkio(number: int) -> int:

    return eval('*'.join(char for char in str(number) if char != '0'))
